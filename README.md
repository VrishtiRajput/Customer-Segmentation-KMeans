# Customer Segmentation using K-Means Clustering

## Overview

This project uses K-Means Clustering to group customers of a retail store based on their purchasing behavior.

## Dataset

Mall Customers Dataset

Features Used:
- Annual Income (k$)
- Spending Score (1-100)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Google Colab

## Methodology

1. Data Loading
2. Data Preprocessing
3. Feature Scaling
4. Elbow Method
5. K-Means Clustering
6. Cluster Visualization

## Results

The customers were successfully segmented into 5 clusters based on spending habits and annual income.
